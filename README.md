# raftLabs-test

# Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/mohammedShafaz/raftLabs-test.git
cd raftLabs-test
```

2. Configure Environment Variables

```bash
cp .env.sample .env
npm install
```

3. Install Dependencies

```bash
npm install
```

4. Run the Application

For Development:

```bash
npm run dev
```

For Production:

```bash
npm run build
```

Start the server:

```bash
npm run start
```
